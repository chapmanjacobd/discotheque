# discotheque

Golang implementation of xklb/library

## Install

    go install github.com/chapmanjacobd/discotheque/cmd/disco@latest

## Usage

### add

Add media to database

<details><summary>Usage</summary>

```bash
$ disco add -h
```

</details>

### check

Check for missing files and mark as deleted

<details><summary>Usage</summary>

```bash
$ disco check -h
```

</details>

### print

Print media information

<details><summary>Usage</summary>

```bash
$ disco print -h
```

</details>

### watch

Watch videos with mpv

<details><summary>Usage</summary>

```bash
$ disco watch -h
```

</details>

### listen

Listen to audio with mpv

<details><summary>Usage</summary>

```bash
$ disco listen -h
```

</details>

### stats

Show library statistics

<details><summary>Usage</summary>

```bash
$ disco stats -h
```

</details>

### history

Show playback history

<details><summary>Usage</summary>

```bash
$ disco history -h
```

</details>

### optimize

Optimize database (VACUUM, ANALYZE, FTS optimize)

<details><summary>Usage</summary>

```bash
$ disco optimize -h
```

</details>

### tui

Interactive TUI media picker

<details><summary>Usage</summary>

```bash
$ disco tui -h
```

</details>

### readme

Generate README.md content

<details><summary>Usage</summary>

```bash
$ disco readme -h
```

</details>

### open

Open files with default application

<details><summary>Usage</summary>

```bash
$ disco open -h
```

</details>

### browse

Open URLs in browser

<details><summary>Usage</summary>

```bash
$ disco browse -h
```

</details>
